Number Systems

In Computer Science, data is represented using different number systems. The most common are Decimal, Binary, Octal, and Hexadecimal.

Number System Base Digits Used Definition
Decimal 10 0–9 The system we use in everyday life; based on ten symbols.
Binary 2 0, 1 The system used internally by computers; uses only two digits.
Octal 8 0–7 Base-8 system; sometimes used as a shorthand form of binary.
Hexadecimal 16 0–9, A–F Base-16 system; commonly used for memory addresses and color codes.

Conversion Steps (Radix to Radix)

1. Decimal → Binary

  1. Divide the decimal number by 2.
  2. Record the remainder (0 or 1).
  3. Use the quotient and divide again by 2.
  4. Repeat until the quotient is 0.
  5. Read the remainders from bottom to top.

Example: 13₁₀ → 1101₂

2. Binary → Decimal

  1. Write the binary number.
  2. Assign powers of 2 from right to left (2⁰, 2¹, 2², ...).
  3. Multiply each bit (0 or 1) by its corresponding power of 2.
  4. Add all the products.

Example: 1101₂ = (1×2³) + (1×2²) + (0×2¹) + (1×2⁰) = 13₁₀

3. Decimal → Octal

  1. Divide the decimal number by 8.
  2. Record the remainder.
  3. Use the quotient and divide again by 8.
  4. Repeat until the quotient is 0.
  5. Read the remainders from bottom to top.

4. Decimal → Hexadecimal

  1. Divide the decimal number by 16.
  2. Record the remainder (0–15). Use A–F for 10–15.
  3. Use the quotient and divide again by 16.
  4. Repeat until the quotient is 0.
  5. Read the remainders from bottom to top.

5. Binary ↔ Octal (Shortcut)

Binary → Octal

  1. Group the binary digits into sets of 3 from right to left.
  2. Convert each group of 3 bits to its octal equivalent.

Octal → Binary: Convert each octal digit to a 3-bit binary value.

6. Binary ↔ Hexadecimal (Shortcut)

Binary → Hexadecimal

  1. Group the binary digits into sets of 4 from right to left.
  2. Convert each group of 4 bits to its hexadecimal equivalent.

Hexadecimal → Binary: Convert each hex digit to a 4-bit binary value.

History of Computers

Computers have evolved through several generations, each introducing new technology and improvements in size, speed, and reliability.

First Generation (1940s – 1950s)

First-generation computers used vacuum tubes as the main electronic component. They were very large, produced a lot of heat, and consumed a lot of electricity. Programming was done in machine language.

Second Generation (1950s – 1960s)

Second-generation computers used transistors instead of vacuum tubes. They were smaller, faster, cheaper, and more reliable. High-level languages like COBOL and FORTRAN were introduced.

Third Generation (1960s – 1970s)

Third-generation computers used integrated circuits (ICs), which combined many transistors on a single chip. This made computers even smaller, more powerful, and more efficient.

Fourth Generation (1970s – Present)

Fourth-generation computers use microprocessors, where thousands or millions of transistors are integrated in one chip. This generation introduced personal computers, laptops, and mobile devices.

Fifth Generation (Present and Beyond)

Fifth-generation computers focus on artificial intelligence (AI), advanced parallel processing, and new technologies such as quantum computing. The aim is to develop systems that can make intelligent decisions.

Application Software

Below are examples of the environments of Microsoft Word, Excel, and PowerPoint. Each image is accompanied by labels and explanations.

Microsoft Word

Labels and Explanations

  • Title Bar: Shows the document name and Word application.
  • Ribbon: Contains tabs (Home, Insert, Layout, etc.) with groups of commands.
  • Quick Access Toolbar: Shortcuts to common commands like Save and Undo.
  • Document Area: Main area where you type and edit your text.
  • Status Bar: Displays information such as page number and word count.

Microsoft Excel

Labels and Explanations

  • Title Bar: Shows the workbook name and Excel application.
  • Ribbon: Contains tabs (Home, Formulas, Data, etc.) with command groups.
  • Formula Bar: Displays and allows editing of the active cell’s content or formula.
  • Worksheet Area (Cells): Grid of rows and columns where data is entered.
  • Sheet Tabs: Switch between different worksheets in the workbook.

Microsoft PowerPoint

Labels and Explanations

  • Title Bar: Shows the presentation name and PowerPoint application.
  • Ribbon: Contains tabs such as Home, Design, Transitions, Slide Show, etc.
  • Slides Pane: Displays thumbnail previews of all slides.
  • Slide Area: Main area where you design and edit the current slide.
  • Notes Pane: Area where you can type speaker notes for each slide.
